## 📌 Project Objective
The goal of this project is to perform Exploratory Data Analysis (EDA) and answer key business questions using SQL to uncover patterns, trends, and actionable insights from Indian E-Commerce transactional data.

This project focuses on:
- Developing strong SQL querying skills
- Understanding data distributions & customer behaviour
- Creating KPI-driven dashboard mockups
- Presenting insights using visualizations

---

## 🗂️ Dataset Description
The dataset contains Indian e-commerce order data (2023–2024) with the following key fields:

- Order Date & Delivery Date
- Revenue & Profit
- Quantity & Unit Price
- Product Name
- Category & Sub-Category
- Customer Name & ID
- City / Region
- Discount
- Payment Mode
- Order Status (Delivered / Returned / Cancelled)
- Segment (Retail / Wholesale / B2B)

**Total Records:** 5,000 orders | **Period:** Jan 2023 – Dec 2024

## 🔍 Key Insights
- **Electronics** category generates the highest revenue overall
- **Beauty** category has the highest profit margin %
- **UPI** is the most preferred payment mode (35%+ orders)
- **High discounts (>20%)** significantly reduce or eliminate profit
- **West region** (Mumbai, Pune, Surat, Ahmedabad) leads in total sales
- **B2B segment** contributes higher average order values
- **Returned orders** are highest in Electronics and Clothing
